# 06-user-manual · 用户手册

本目录存放 `ammm56/amcontrol` 的**用户手册**，面向设备现场的操作员（Operator）和工程师（Engineer），提供界面操作、设备参数配置和报警处理的完整指引。

---

## 内容范围

| 章节 | 说明 |
|------|------|
| 登录与权限 | 登录流程、角色说明（操作员/工程师/管理员）、密码修改方式 |
| 主界面导航 | 左侧两级导航结构说明、各页面功能简介 |
| 运动控制监视 | 轴状态查看（脉冲位置/规划毫米位置/编码器毫米位置）、回零操作、点动操作 |
| IO 监视 | DI/DO 状态查看、磁贴卡片说明、按控制卡筛选方法 |
| 执行器控制 | 气缸/夹爪/真空阀的手动控制操作 |
| 报警处理 | 报警抽屉使用方式、报警详情查看、报警确认与处理流程 |
| 设备参数配置 | 轴参数配置页面操作、IO 映射配置（工程师及以上权限） |
| 用户管理 | 用户新增/编辑/删除（管理员权限）、权限分配（页面级） |

---

## 使用说明

- 操作员（Operator）：可查看运动监视、IO 监视信息；不可修改参数或执行调试操作。
- 工程师（Engineer）：可进行参数配置和手动控制，不可进行用户管理。
- 管理员（Am）：拥有全部权限，包括用户管理和权限配置。
