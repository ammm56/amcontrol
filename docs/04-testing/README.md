# 04-testing · 测试文档

本目录存放 `ammm56/amcontrol` 项目的**测试相关文档**，包括测试策略、测试用例模板和调试指引。

---

## 内容范围

- **测试策略**：项目整体测试方法，说明哪些层采用单元测试、集成测试以及虚拟卡替代测试。
- **虚拟卡调试指引**：如何使用 `VirtualMotionCardService` 替代真实运动控制卡进行功能验证，包括：
  - 虚拟卡与真实卡的切换配置方式；
  - 如何触发虚拟 DI 信号以推进流程；
  - 与 3D 仿真前端联调时的注意事项。
- **测试用例模板**：描述运动控制、IO、PLC 通信等关键业务流程的测试用例格式。
- **集成测试说明**：涉及多模块联动（Motion + IO + 流程）的集成测试注意事项。
- **已知测试局限**：无真实硬件环境下的测试边界说明。
