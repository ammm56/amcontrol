# 07-release-notes · 版本发布记录

本目录按版本号归档 `ammm56/amcontrol` 的**版本变更日志**，记录每次发布的新增功能、变更说明、缺陷修复和升级注意事项。

---

## 命名规范

版本记录文件命名格式：

```
release-v<主版本>.<次版本>.<补丁>.md
```

示例：

```
release-v1.0.0.md
release-v1.1.0.md
```

---

## 版本记录模板

每个版本文件应包含以下内容：

```markdown
# v<版本号> 发布记录

**发布日期**：YYYY-MM-DD  
**发布类型**：正式版 / 预发布 / 热修复

## 新增功能

## 功能变更

## 缺陷修复

## 破坏性变更

## 升级说明
```
