# 00-governance · 文档治理

本目录定义 `ammm56/amcontrol` 项目的**文档治理规范**，包括命名约定、元数据标准、目录结构规则、长期维护规则和索引使用方式等内容。

所有文档贡献者在新建或修改文档前，请先阅读本目录下的规范文件。

| [document-maintenance-rules.md](document-maintenance-rules.md) | 文档长期维护规则（事实来源优先级、实现/规划文档并存规则、索引维护方式、同步清单） |

## 适用范围

本目录中的规范适用于 `Docs/` 目录下的所有文档，以及项目各子模块中的 `README.md` 文件和需要与 `Docs/` 同步维护的仓库级说明文件。
